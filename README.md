Brota é uma plataforma web criada para apoiar pequenos agricultores e produtores da agricultura familiar na tomada de decisões sobre plantio. A primeira versão do projeto recomenda culturas com base em temperatura, solo e umidade. Futuramente, a plataforma poderá integrar dados climáticos, preços de sementes, fornecedores, conteúdos educativos e ferramentas comunitárias para fortalecer a produção local.
